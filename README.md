# Mini-Projek-PP-Kelompok-5

Kia adalah seorang mahasiswa baru Ilmu Komputer yang memiliki jadwal padat setiap harinya. Ia harus mengatur waktu untuk kuliah, mengerjakan tugas, mengikuti organisasi, dan beristirahat.
